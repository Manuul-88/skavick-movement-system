# Skavick - Movement System (UE5)

Sistema de movimiento avanzado desarrollado en Unreal Engine 5.

## Características

- Climbing system (paredes y techo)
- Movimiento dinámico basado en superficies
- Rotación del personaje según normal de la superficie
- Implementación en C++ con soporte para Blueprints

## Tecnologías

- Unreal Engine 5
- C++
- CharacterMovementComponent personalizado

## Descripción

Este sistema permite al personaje desplazarse libremente en superficies verticales y techos, simulando alteraciones de gravedad.
